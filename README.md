# LeagueSync Admin

LeagueSync Admin is the desktop app tournament staff use to run a league day-to-day — manage teams and players, build the schedule, and enter live scores that spectators see update in real time on the [LeagueSync website](https://sporting-leagues-web-fhbk.vercel.app).

## Why LeagueSync

Running a tournament off spreadsheets and word-of-mouth means scores lag, standings go stale, and someone always has to shout across the gym to ask what the score is. LeagueSync fixes that: whoever's sitting at the scorer's table taps a button, and the score, clock, and leaderboard update live for every spectator watching on their phone — no refreshing, no asking around.

## Features & Benefits

- **Live scoring** — tap in points per player as the game happens; spectators see it within seconds.
- **Match clock with pause/resume** — a single button pauses and resumes the game clock (injuries, timeouts, halftime overruns) without throwing off the timing.
- **Teams & players management** — add/remove teams and rosters for any league, any time.
- **Flexible scheduling** — set up matches by stage (Group Stage, Knockout, Quarter/Semi/Finals) with date and time.
- **Cards & discipline tracking** — issue and undo yellow/red cards where the sport uses them.
- **Multi-sport support** — basketball, football, and cricket out of the box, each with sport-appropriate scoring.
- **Multi-league management** — run several tournaments/seasons from one app and switch between them.
- **Automatic champions & leaderboard** — the Champions podium and top-scorer leaderboard are computed automatically as matches finish, no manual tallying.
- **Excel export** — download a single .xlsx per league covering teams/standings, players, match results, the leaderboard, and champions — a handy backup or way to share results outside the app.
- **LED screen display** — a public slideshow page (standings, upcoming matches, announcements, scrolling sponsor strip) built for a venue TV or projector, managed from the app's LED Screen tab.
- **Auto-updates** — the app checks for new versions and updates itself, so you're always on the latest release.
